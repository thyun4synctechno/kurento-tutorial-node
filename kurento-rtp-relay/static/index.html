Tutorial xx: RTP Video Relay

This application shows how to connect a WebRtcEndpoint to a RtpEndpoint. To run this demo follow these steps:

  1. Make sure you have H264 support in your Kurento Media Server machine (package openh264-gst-plugins-bad-1.5) .
  2. Open this page with a browser compliant with WebRTC (Chrome, Firefox).
  3. Update the RTP's destination IP address and Port to point to a valid destination.
  4. Click on the Start button.
  5. Grant access to the camera and microphone. After the SDP negotiation the RTP relay should start.
  6. In the destination machine, open any player that can proccess RTP streams. Let's use VLC media player.
    • Copy/paste the Session Descriptor from the textarea into a file called input.sdp
    • In VLC, open the input.sdp file (Media > Open File) and wait for the video to be played.
  7. Click on Stop to finish the communication.
